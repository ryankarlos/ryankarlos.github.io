Latent Dirichlet Allocation (LDA) is a topic modelling technique, used to classify text in a document to a particular topic. It builds a topic per document model and words per topic model, modeled as Dirichlet distributions.

Below is a visualisation of the output of the LDA model for 5 topics fitted using the Python package gensim. The visualisation was generated using the LDAvis package. Hover around the different topics (circles) and check the distribution of words in the right panel.
